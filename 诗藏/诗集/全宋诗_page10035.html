湖上喜明霁，寻春偶啸俦。 花多红夹马，山远翠藏楼。 水气烟中合，风光草际流。 都人待千骑，快作谢公游。 元绛　和人 笔下云霞落彩牋，诸君清思尽佳篇。 谁人读得郊居字，流水汤汤正满弦。 元绛　和上巳西湖胜游 清明风日战方酣，朱毂雍容旆鬺。 火令正严绵上俗，禊游仍胜会稽岩。 马盘金埒连乾湿，人映氊车帽角搀。 湖水绿烟浮醉席，杏花红雨拂春衫。 公如庾老情非浅，客似渊君调不凡。 只恐颍川飞召节，便随五凤降绨函。 元绛　和圣徒洛中九老会 五日佳辰郡政闲，延宾谈笑豁幽关。 阊门歌舞尊罍上，林屋烟霞指顾间。 德应华星临颍尾，年拘皓发下商颜。 名花美酒疏钟永，坐见斜晖隐半山。 元绛　贺王禹玉 前殿听宣中禁制，南宫看集外朝班。 星辰影落三阶下，桃李阴成四海间。 元绛　怀荆南旧游 去年曾醉海棠丛，闻说新枝发旧红。 昨夜梦回花下饮，不知身在玉堂中。 元绛　将去郡席上呈同僚 在官七百二十日，只为台州不为身。 民事勤劳心力尽，世涂忧畏鬓华新。 离筵欲醉樽前月，美景难留洞口春。 他日相逢应更好，诸君皆是眼中人。 元绛　句 笙歌车马东方晓，风月山川北固凉。 元绛　句 英翰五五出南闽。 元绛　句 小雨疏疏向晚晴。 元绛　句 仙骥籬云穿仗下，佛花吹雨匝天流。 元绛　句 五日看花怜并叶，今朝斗草得宜男。 元绛　句 五蓂开端荚，百草斗香苕。 元绛　句 丝竹渐高桡鼓急，瑶津亭下竞凫车。 元绛　句 使君麾盖傲烟霞。 元绛　句 九重侍从三明生，四纪乾坤一老臣。 元绛　句 神石溪头好明月，夜来谁此凭兰桡。 元绛　句 玉露精神山合沓，溪成文字水横斜。 元绛　句 黄阁势连东凤阙，紫枢光直右银台。 元绛　句 联名适以三株树，传玩惊看五朵云。 元绛　句 花阴转午清风细，玉燕钗头艾虎轻。 元绛　句 陈前舆服同桓傅，拜后金珠有鲁公。 元绛　句 可惜闽州风物好，一生魂梦到三山。 元绛　句 何人更得通天带，谋合君心只晋公。 元绛　句 菖酒朝觞满，兰汤晓浴温。 元绛　句 过庐都失眼前人。 元绛　流觞亭诗 春风流水出岩扉，载酒轻舟飏彩旗。 把得玉杯还自笑，老翁真令似童儿。 元绛　诗一首 闽山过尽见吴山，天载天涯五马还。
