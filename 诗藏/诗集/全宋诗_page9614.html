幻松作璧璧作梅，豪气勃郁尚不开。 琼艘满釂梅雪下，吐出西湖有声画。 杨万里　芗林五十咏　白鹤亭 可怜林下鹤，独肯伴先生。 犹有冲天志，君听叫月声。 杨万里　芗林五十咏　百花洲 茂苑犹香名，南阳已荒皁。 更兼芗林中，三洲是三岛。 杨万里　芗林五十咏　北垞 住处何曾远，林间别是凉。 清风酣一枕，底许觅羲皇。 杨万里　芗林五十咏　碧芦淙 春笋肥堪菜，秋花暖可毡。 何须双野鸭，屏上五湖天。 杨万里　芗林五十咏　碧梧 往往游西崦，时时憩午阴。 凤来人不见，飞入碧枝深。 杨万里　芗林五十咏　丛桂 不是人间种，移从月胁来。 广寒香一点，吹得满山开。 杨万里　芗林五十咏　东皋 避世无功酒，临流元亮诗。 鹤鸣入天听，叶下报秋悲。 杨万里　芗林五十咏　读书堂 老以书为命，眼将书作雠。 芗林与诚叟，同贩短檠愁。 杨万里　芗林五十咏　鹅篆滩 不傅右军字，且留房相池。 新雏黄似酒，把酒看群嬉。 杨万里　芗林五十咏　泛宅 犹道山中浅，仍移水上居。 俗人又剥啄，掉入白芙蕖。 杨万里　芗林五十咏　放鱼矶 为惜卅砂尾，亲投碧草潭。 前头有香饵，千万不须贪。 杨万里　芗林五十咏　芙蓉沜 岸植木菡萏，池栽水拒霜。 那知一家种，同艳不同香。 杨万里　芗林五十咏　柑子园 秋颗酣天酒，春花噀国香。 芗林亲手种，中有洞庭霜。 杨万里　芗林五十咏　瓜田 独酌圣贤酒，新尝子母瓜。 丁宁林下友，莫道故侯家。 杨万里　芗林五十咏　皈来桥 已赓彭泽辞，更拟辋川诗。 未老还山了，犹嫌皈较迟。 杨万里　芗林五十咏　过花桥 北垞通南埭，前花映後花。 雨多桥板断，折碎两涯霞。 杨万里　芗林五十咏　海棠洞 艳翠春销骨，妖红醉入肌。 花仙别无诀，一味服燕支。 杨万里　芗林五十咏　黑坳 坳中一池墨，腕中百斛力。 洒作五十诗，千年照泉石。 杨万里　芗林五十咏　槐阴墿 阴作官街绿，花催举子黄。 公家有三树，犹带凤池香。 杨万里　芗林五十咏　驾月桥 紫霓横天度，银盘涌水来。 先生骑霓脊，盘上卷金杯。 杨万里　芗林五十咏　菊坡 西风破幽馥，东篱散清好。 渊明不可作，解后芗林老。 杨万里　芗林五十咏　兰畹 健碧缤缤叶，斑红浅浅芳。 幽香空自秘，风肯秘幽香。 杨万里　芗林五十咏　梨花原 禁火晓未暖，蹈青昏恰皈。 何堪一原雪，将冷入春衣。 杨万里　芗林五十咏　连辉观 极目斜阳树，乘风落月梁。 更和山与水，并作一秋光。 杨万里　芗林五十咏　梅坡
