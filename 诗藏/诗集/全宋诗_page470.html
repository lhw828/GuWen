不善临吾前，真如负芒刺。 陈淳　閒居杂咏三十二首　笃行 学问思辨者，於中莹且精。 欲实据而有，要在笃於行。 陈淳　閒居杂咏三十二首　耳 耳所以司听，听正乃为聪。 匪彝谨勿闻，闻之则为聋。 陈淳　閒居杂咏三十二首　夫妇 夫妇亦大端，乾男而坤女。 一言在有别，不可欲败度。 陈淳　閒居杂咏三十二首　父子 父子本天性，人伦此其大。 一言在有亲，不可薄厥爱。 陈淳　閒居杂咏三十二首　改过 过者动之差，毋容实诸已。 才觉必速改，乃不为吾累。 陈淳　閒居杂咏三十二首　谨思 论学取诸友，举隅发之师。 欲自得其传，要在谨厥思。 陈淳　閒居杂咏三十二首　君臣 君臣本大分，天尊而地卑。 一言在有义，不可以为利。 陈淳　閒居杂咏三十二首　口 口所以司言，所言必正说。 非法谨勿道，道之为噬噎。 陈淳　閒居杂咏三十二首　礼 礼者人之门，节文自中根。 所主一以敬，出入无不存。 陈淳　閒居杂咏三十二首　礼维 礼以维其心，在心无不敬。 非此勿言动，非此勿视听。 陈淳　閒居杂咏三十二首　廉维 廉以维其心，在心常有辨。 一介不妄取，真如视土粪。 陈淳　閒居杂咏三十二首　隆师 师者人之范，辨惑正吾疑。 苟不就有道，伥伥其何之。 陈淳　閒居杂咏三十二首　明辨 利与善之间，微似未易断。 欲无毫厘差，要在明厥辨。 陈淳　閒居杂咏三十二首　目 目所以司视，视正乃为明。 非礼谨勿睹，睹之则为盲。 陈淳　閒居杂咏三十二首　朋友 朋友同门志，所以辅吾仁。 一言在有信，不可私狎亲。 陈淳　閒居杂咏三十二首　迁善 善者性所有，不可无诸躬。 每见必勇迁，吾德乃可崇。 陈淳　閒居杂咏三十二首　亲友 友者人之辅，以善相切磨。 不取直谅闻，其如损德何。 陈淳　閒居杂咏三十二首　仁 仁人之安宅，在心本全德。 要常处於中，不可违终食。 陈淳　閒居杂咏三十二首　审问 物理本不齐，杂然各异分。 参伍宁无疑，要在审所问。 陈淳　閒居杂咏三十二首　手 手所以司执，所执必正事。 回德谨勿持，持必为痿痺。 陈淳　閒居杂咏三十二首　悌 悌以其事兄，温恭尽乃情。 出门惟敬长，内外一於诚。 陈淳　閒居杂咏三十二首　孝 孝以事其亲，斯须不离身。 始终惟爱敬，二者在书绅。 陈淳　閒居杂咏三十二首　心 心为形之君，所主一身政。 持养常清明，百体皆顺令。 陈淳　閒居杂咏三十二首　信 信以实诸言，於外无妄宣。 要须循尔物，何可背其然。 陈淳　閒居杂咏三十二首　兄弟
