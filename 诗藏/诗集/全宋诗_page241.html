曹勋　山居杂诗九十首 凿池固不广，纳纳隘所居。 苟无水光寒，便觉非吾庐。 漪涟漾芳渚，澄影浮晴虚。 乐哉濠梁心，已复游嘉鱼。 曹勋　山居杂诗九十首 道人范子珉，隐迹朝市端。 笔墨作峰岭，气象争巑屼。 稍变烟云态，便觉风雨寒。 鹤举忽冲霄，大罗得遐观。 曹勋　山居杂诗九十首 闲居养僮仆，衣食必鲜美。 小则爱睡眠，大者趋博戏。 不然言疾病，莫能承付委。 长须与赤脚，昔人诚有理。 曹勋　山居杂诗九十首 范仙最所长，画牛诚绝笔。 围模大小定，毛孔不少失。 膊腕与腹胯，一牛动数日。 惟我颇得之，卧起尽纤悉。 曹勋　山居杂诗九十首 莫怪游春频，只被老逼身。 念此乐天语，诚哉辞甚真。 遵行我不倦，治具当勿嗔。 持炬接迎我，村醉欹冠巾。 曹勋　山居杂诗九十首 三春君卉盛，在花茂与残。 好花正穠丽，不作十日看。 蓓嫩便须赏，开时惟可叹。 继自喜新绿，吾心常并观。 曹勋　山居杂诗九十首 寒毬格虽下，春事亦可寻。 故园少见之，花叶均来禽。 芳蓓暖苞玉，半腮红浅深。 与客屡清赏，只恐风雨侵。 曹勋　山居杂诗九十首 随春动草木，微和入池阁。 群芳争妍媸，红翠媚寥廓。 一枝姑到眼，万态便如昨。 春深风雨来。固已任开落。 曹勋　山居杂诗九十首 煴室宿火销，縜袍良独暖。 病身怯夜永，老怀苦昼短。 栖鸟惊檐头，朝日上窗眼。 未问粥面花，地黄酒须满。 曹勋　山居杂诗九十首 侵晨盥栉起，爽气随凉飚。 曳杖踏露草，不觉路已遥。 水清出沙嘴，烟白横山腰。 致爽列群岫，不待西山招。 曹勋　山居杂诗九十首 闲居必家酿，不以空乏废。 小槽压春碧，浑含揖和气。 三杯御客寒，一卮款仆意。 宁论有贤圣，且复供村醉。 曹勋　山居杂诗九十首 小圃养春色，乘时不可缓。 翌锄常于于，成趣须旦旦。 花摘去年种，喜供一时玩。 举杯付歌儿，野服更萧散。 曹勋　山居杂诗九十首 从昔漫浪心，家居意不适。 晨圾香火余，饭了办游历。 或访僧掇斋，或隐者对奕。 归来任侵灯，龙吟喷霜笛。 曹勋　山居杂诗九十首 随分作栏槛，松竹带花柳。 四时面面看，百种日日有。 玩目必群卉，洗心付芳酒。 吾侪不改常，居閒简可久。 曹勋　山居杂诗九十首 前村有经社，香火讯屡通。 携朋闲步往，老人颜有童。 一杯苦涩茗，满塘蒲稗风。 笑顾纸上佛，何时供渠侬。 曹勋　山居杂诗九十首 老来参同事，勤增不敢失。 人言炉偃月，吾意炉藏日。 朝阳升西山，异液甘若蜜。 一体奠所居，旦旦享清逸。 曹勋　山居杂诗九十首 平生不能棋，遇棋亦輙喜。 从来号万户，知生不救死。 多见国手说，得先乃妙理。
