曹勋　山居杂诗九十首 墙里种桃杏，新春亦已花。 墙外绕扶疏，绿荫皆桑麻。 今年风雨时，日见田稻嘉。 长瓶泻黄汤，且饷田畯茶。 曹勋　山居杂诗九十首 闲里心情拙，还知兴自嘉。 饥鸦时立槛，野蝶自寻花。 客至聊从容，僧来不厌茶。 晚烟时伫立，列岫映余霞。 曹勋　山居杂诗九十首 清晓看田去，乘露衣襟凉。 已有耘夫出，揖我喜非常。 为言苗正好，喜颜笑亦长。 虽有吏在门，亲苗端可偿。 曹勋　山居杂诗九十首 人生行乐尔，行乐则心闲。 尽既无所用，和气滋容颜。 卫生有此妙，余事皆可删。 若益兼忘理，敢告知识间。 曹勋　山居杂诗九十首 松根有至药，琥珀与茯苓。 琥珀未易致，茯苓选宜精。 制与鞠尘细，色如截肪并。 一一伴朝粥，地黄安可京。 曹勋　山居杂诗九十首 招客图客欢，不图能病客。 既醉失匕箸，吐茵堕巾帻。 何堪款曲意，贻念风月夕。 拂旦往问讯，惭恨见颜色。 曹勋　山居杂诗九十首 松色荫人居，松力治人病。 丹砂含其滋，复何延性命。 幽居绕苍髯，不畏夏日盛。 风来听笙竽，便有华胥兴。 曹勋　山居杂诗九十首 春雪翳重霄，寒雨连日夜。 不辍兽炭炽，甚恐庭花谢。 疏帘莫御冷，炉焚屡成灺。 寒食数日间，春事求诸野。 曹勋　山居杂诗九十首 吾年逾七十，真欲原春迟。 岂谓上元过，适丁多雨时。 韶华荏苒半，清明旦夕期。 流莺与蜂蝶。冷落殊未知。 曹勋　山居杂诗九十首 卜居得幽胜，景物无升沉。 峰峦环郛郭，松竹排园林。 时时破寂寥，悠悠赖鸣禽。 於焉自有得，天籁皆余音。 曹勋　山居杂诗九十首 客寄辇毂下，驰心田亩间。 君恩重山岳，臣节殊安闲。 虽幸米盐足，其如尘土颜。 终负告归罪，判然诚独难。 曹勋　山居杂诗九十首 朼去天台居，邑有万户酒。 一出尝十家，已觉醉屡肘。 留连多好事，方圆烦饤斗。 约趁黄昏归，不知月挂柳。 曹勋　山居杂诗九十首 年来心目昏，稍息文字业。 晓拥台山云，时读修禊帖。 稚子来捋须，幼孙每持镊。 晚来微雨休，幽径无不涉。 曹勋　山居杂诗九十首 老去情怀懒，修然只此翁。 闲窗千里月，修竹一帘风。 事与孤鸿远，心将止水同。 故人如问讯，痴坐正观空。 曹勋　山居杂诗九十首 寂历秋风晚，闲庭落叶深。 天高晴日澹，山净暮烟沉。 疾病无时已，年华有底侵。 两行南去雁，一片北归心。 曹勋　山居杂诗九十首 三伏暑可畏，贫富方共之。 雨忽兼飘风，凉与清气随。 一饷涤炎暍，万里失蒸炊。 大哉天地德，好生常所施。 曹勋　山居杂诗九十首 北归了无日，先垄扫何时。 愁与燕鸿远，心同楚客悲。
