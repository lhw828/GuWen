释智遇　偈颂二十四首 虎憎鸡嘴，闻者畏之。 后人衡鉴不高，唤作二俱弄险。 释智遇　偈颂二十四首 人间无，天上有， 往往无人脱窠臼。四海娟娟洗玉魂， 九野茫茫白兔走。寒山子， 不关口，也落马驹群队后。 释智遇　偈颂二十四首 一人要贫贫不得，一人要富富不得。 要知贫富不相当，且请各归本位立。 释智遇　偈颂二十四首 朕闻上古，其风朴略。 王言如丝，谁敢不听。 忽有人问鄮山，只向他道， 风门海口，当风抵浪， 也须是个人始得。 释智遇　偈颂二十四首 镜容鹰爪，面目可憎。 掉发南岳，下山教化。 从臾檮林，对御讲经。 不守疆界，干犯清平。 灭得中峰正法眼，破沙盆子话方行。 释智遇　偈颂二十四首 旧去新来懒送迎，岩间冢下见枯形。 忍成残腊半宵梦。侩地寒檠两岁灯。 释智遇　偈颂二十四首 时迁物换，革故鼎新。 土膏未动，商量打春。 太公有意垂钓，夫子无心获麟。 释智遇　偈颂二十四首 去时晓露消袢暑，归日秋声满夕阳。 恩渥重重何以报，望无云处祸天长。 释智遇　偈颂二十四首 一年又一年，循环数不足。 本分面上人，犹如隔罗縠。 惟有南极老人，扣天鼓三下， 望北关而祝，愿我王万福。 释智遇　偈颂二十四首 外阖不闭，跨天下而无蕲。 会得许你，升其堂入其室。 释智遇　偈颂二十四首 世尊三昧，金口玉音， 亲付嘱紫芝检。九重城里凤衔来， 再沾雨露之恩，光阐法门之盛。 声前妙证，丛动群心。 释智遇　偈颂二十四首 烟惨淡，石玲珑。 面面厮觑，千峰万峰， 一团和气在其中。 释智遇　偈颂二十四首 春入湖山，先放花明。 御苑人归上国，南山鹤唳青公。 壮皇都人杰地灵，窥紫府洞天风月。 释智遇　偈颂二十四首 绝类离伦。不妨垂手入廛， 毕竟难逃至化。且望阙酬恩， 如何祝替。版图远奏尧天阔， 万物呈祥乐圣情。奸悄互队， 对面千里。有人寄僧过夏， 南山大开东合，彼此出家儿。 释智遇　偈颂二十四首 此山无路，及门者谁。 会得摆手同归，不然随我来。 释智遇　偈颂二十四首 释迦掩室於摩竭，因甚坐在者里。 炷拜勤渠，齐之以礼。 释智遇　偈颂二十四首 虎头燕颔，鸟嘴鱼腮。 尽向者里纳款，且道者里是甚麽所在。 释智遇　偈颂二十四首 有意退栏，无心耕牧。 九重降勑，如万国春回。 一道恩光，似千百并照。 法门腾焕，岩壑生秋。 释智遇　偈颂二十四首 日出而作，日入而息。 熙熙然如登春台，且道承谁恩力。 释智遇　偈颂二十四首 出山而见，入户而知。
